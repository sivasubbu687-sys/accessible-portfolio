# Siva's Accessible Portfolio

Welcome to my personal portfolio repository! 

🌍 **Live Website:** [https://sivasubbu687-sys.github.io/accessible-portfolio/](https://sivasubbu687-sys.github.io/accessible-portfolio/)

## About
This is an accessible, responsive, and beautifully designed web portfolio built with Semantic HTML, CSS Grid, and Vanilla Javascript. It features a fully functioning contact form powered by Web3Forms.

## Features
- **Live Contact Form**: Sends messages directly to my inbox without needing a backend server.
- **Dark/Light Mode**: Full theme switching support.
- **Accessibility**: ARIA labels, semantic markup, and full keyboard navigation.
- **Responsive**: Fully responsive CSS Grid and Flexbox layout for mobile and desktop.
